The Web — Network Visualization

Visual expression of Branch 01 · 61 nodes · 7 relationship types
→

The force-directed graph for Branch 01. A physics-simulated map of every actor in the cronyism network — cabinet, donors, family, lobbyists, captured oversight bodies, legislation, vendors, and public-cost endpoints. Animated particles trace money as it moves between donation and contract edges. Drag any node, click for the full sourced record, and filter in two dimensions.

What you can do

  • Hover any node for the top-line tooltip with its primary source.
  • Click any node to jump to its full write-up card below the graph — every card carries description + source citation.
  • Search across all names, dollar amounts, and descriptions at once.

The record

Evidence standard & methodology

What this project is

Documentation of a governance pattern where public power is systematically aligned with donor and personal interest. Every claim ties to a primary source: court filings, land titles, Elections Alberta records, the Lobbyist Registry, the Alberta Gazette, Hansard, FOIP releases, or sustained reporting from the Globe and Mail, CBC, The Tyee, and The Breakdown.

Allegations are flagged as allegations. Counter-evidence is acknowledged up front on every claim. Inference is not presented as fact. Legitimate grievances are conceded first — credibility depends on acknowledging what is real before dismantling what is exaggerated.

What this project is not

Not a criminal indictment. Nothing here asserts that any named actor is criminally prosecuted or criminally provable beyond reasonable doubt. Not a harassment tool. Pseudonymous accounts in The Ledger stay pseudonymous — no unmasking, role-level attribution only.

Best viewing

Interactive visualizations (The Roadmap, The Web — Network Visualization) work best in a desktop browser (Chrome, Firefox, Safari). They work on mobile too — drag and pinch as you'd expect — but the dense network views are easier to read on a larger screen. If you receive one of these as an email attachment on iPhone, tap “Open in Safari” or save to Files first; Mail's inline preview blocks the scripts that make them work.
